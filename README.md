# Minesweeper
Реализация [игры «Сапёр»](https://ru.wikipedia.org/w/index.php?title=Сапёр_(игра)) в варианте с шестиугольными клетками, а также решателя игры.
## Уровни сложности
В зависимости от уровня сложности на поле выставляется определенный процент мин:
* Простой: 10 %
* Средний: 15 %
* Сложный: 18 %
* Очень сложный: 25 %
